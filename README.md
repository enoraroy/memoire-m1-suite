# memoire-m1-suite
Des données supplémentaires pour compléter le dépôt de mon mémoire.

Si les scripts sont hétérogènes dans leurs sources, mêlant à la fois du MongoDB et du MySQL, cela s'explique principalement par le fait que ma base a été migrée en cours d'année, et que mes égarements intellectuels n'ont pas attendu la migration de base pour se concrétiser en scripts. Cela ne change fondamentalement rien, puisqu'il s'agit de données qui se trouvent au final dans MongoDB (rien n'est donc perdu!).
